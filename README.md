# Тренер — Wear OS (Galaxy Watch 7)

Самостоятельное приложение для Galaxy Watch 7 (Wear OS). Работает на часах без телефона:

- 🗓️ Та же 12-недельная программа для новичка (день A / день B, прогрессия по неделям)
- 🔥 Разминка в начале (5 движений с таймером и вибрацией)
- ⏱️ Таймер отдыха между подходами с вибрацией (+15 сек / пропустить)
- ✅ Счётчик подходов, переход по упражнениям, финиш
- 💾 Прогресс (номер тренировки/недели) хранится на часах

## Сборка через GitHub Actions (без Android Studio)

1. Создай новый репозиторий на GitHub и залей туда эту папку `fit-watch`.
2. Открой вкладку **Actions** — workflow «Build Wear OS APK» запустится сам (или нажми **Run workflow**).
3. Когда сборка закончится (~3–5 мин), внизу страницы запуска скачай артефакт **fit-watch-debug-apk** → внутри `app-debug.apk`.

## Установка на Galaxy Watch 7

1. На часах включи **Настройки → О часах → Сведения о ПО** → тапни «Номер сборки» 7 раз → появится **Режим разработчика**.
2. В **Настройках разработчика** включи **Отладку по ADB** и **Отладку по Wi-Fi** (Debug over Wi-Fi). Запомни IP часов.
3. На компьютере с установленным `adb`:
   ```
   adb connect <IP_часов>:5555
   adb install app-debug.apk
   ```
4. Приложение **«Тренер»** появится в списке приложений на часах.

## Стек
Kotlin · Jetpack Compose for Wear OS · Gradle 8.7 · AGP 8.5.2 · minSdk 30 (Wear OS 3+).

## Структура
- `app/src/main/java/com/fitcoach/watch/MainActivity.kt` — вся программа и UI
- `.github/workflows/build.yml` — облачная сборка APK
